03.Write a script that finds the maximal sequence of equal elements in an array.

04.Write a script that finds the maximal increasing sequence in an array. Example: {3, 2, 3, 4, 2, 2, 4}  {2, 3, 4}.

05.Sorting an array means to arrange its elements in increasing order. Write a script to sort an array. Use the "selection sort"

move it at the second position, etc. Hint: Use a second array

06.Write a program that finds the most frequent number in an array. Example: {4, 1, 1, 4, 2, 3, 4, 4, 1, 2, 4, 9, 3}  4 (5 times)
